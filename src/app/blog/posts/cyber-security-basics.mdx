# Memulai Perjalanan di Cyber Security

Cyber security bukan cuma soal hacking. Ini adalah bidang yang luas yang mencakup perlindungan data, sistem, dan jaringan.

## CIA Triad
Konsep paling dasar yang harus kamu tahu adalah CIA Triad:
1. **Confidentiality**: Menjaga kerahasiaan data.
2. **Integrity**: Menjaga keaslian data agar tidak diubah orang tak berwenang.
3. **Availability**: Memastikan data dan sistem selalu bisa diakses saat dibutuhkan.

## Jalur Belajar
Jika kamu tertarik, kamu bisa mulai dari:
- **Networking**: Pahami cara kerja internet (TCP/IP, DNS, HTTP).
- **Linux**: Pelajari command line, karena kebanyakan tools security jalan di Linux.
- **Web Security**: Pelajari bug-bug umum seperti SQL Injection atau XSS.
